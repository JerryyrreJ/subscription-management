# Subscription Manager

Subscription Manager 用来管理周期性订阅、续费日期、支出趋势和分类统计。

应用默认是本地优先模式。不配置任何托管服务时，数据会保存在浏览器里。需要账号、云同步、付费激活、服务端提醒或自动化时，可以按需启用 Supabase、Stripe、Netlify Functions、Bark 和开放 API。

## 可以做什么

- 记录订阅名称、分类、金额、币种、计费周期和续费日期。
- 按基础币种查看月度和年度支出。
- 在 CNY、USD、EUR、JPY、GBP、AUD、CAD、CHF、HKD 和 SGD 之间换算。
- 按名称、金额、续费日期、分类或创建时间排序筛选。
- 管理内置分类和自定义分类。
- 通过 JSON 导入和导出订阅数据。
- 启用 Supabase 账号登录和多设备同步。
- 使用 Bark 推送续费提醒。
- 创建 Developer API Key，通过开放 API 做订阅 CRUD 自动化。

