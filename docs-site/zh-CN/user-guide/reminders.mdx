# 续费提醒

续费提醒可以在订阅再次扣费前提醒你。

## 你需要什么

续费提醒使用 Bark。你需要：

- Bark server URL。
- Bark device key。
- 提醒时间，例如提前几天提醒。

如果使用托管版本，提醒功能还需要部署方启用。提醒不可用时，你仍然可以在应用里手动查看续费日期。

## 配置提醒

1. 打开 **设置**。
2. 进入 **通知**。
3. 粘贴 Bark URL，或填写 Bark server 和 device key。
4. 选择提醒时间。
5. 保存设置。

应用可以从 Bark 示例 URL 中自动提取 device key。

## 单个订阅开关

每个订阅都有自己的通知设置。不想收到某个订阅的提醒时，可以关闭该订阅的通知。

## 排查提醒问题

如果没有收到提醒：

- 检查 Bark server URL 是否可访问。
- 检查 device key 是否正确。
- 检查该订阅是否启用了通知。
- 检查下一次付款日期是否有效且在未来。
- 如果托管提醒依赖云同步，检查你是否已登录。
