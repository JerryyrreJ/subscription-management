# 导入导出

导入导出适合备份数据，或在浏览器、设备、部署之间迁移数据。

## 导出数据

1. 打开用户菜单，或进入 **设置** > **通用**。
2. 选择 **导出数据**。
3. 妥善保存下载的 JSON 文件。

导出文件包含应用后续可以导入的订阅和分类数据。

## 导入数据

1. 打开用户菜单，或进入 **设置** > **通用**。
2. 选择 **导入数据**。
3. 选择之前导出的 JSON 文件。
4. 查看导入预览。
5. 确认导入。

导入流程会先校验文件，再应用变更。

## 保护备份文件

导出文件可能包含订阅名称、付款金额、续费日期、分类和通知偏好。请像个人财务数据一样保管。

## 什么时候导出

- 更换浏览器或设备前。
- 清理浏览器数据前。
- 测试自部署版本前。
- 进行大量编辑前。
