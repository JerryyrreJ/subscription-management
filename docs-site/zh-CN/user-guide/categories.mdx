# 分类

分类可以帮助你整理订阅，也会影响筛选和支出分析。

## 选择分类

添加或编辑订阅时，从分类列表里选择一个分类。分类名称会显示在筛选器、图表和订阅卡片里。

## 添加自定义分类

1. 打开 **设置**。
2. 进入 **分类**。
3. 添加新的分类名称。
4. 在添加或编辑订阅时使用它。

你也可以在订阅表单选择分类时直接添加新分类。

## 调整顺序

打开 **设置** > **分类**，拖动分类到你喜欢的顺序。这个顺序会影响分类选择器中的显示顺序。

## 删除或恢复

你可以删除自定义分类，也可以在分类设置中恢复默认分类。如果某个分类正在被订阅使用，请仔细阅读确认提示，避免后续难以整理。

## 建议

- 分类名尽量简短统一。
- 可以使用 Streaming、Productivity、Cloud、Finance、Shopping 这类宽泛分类。
- 更推荐按用途分类，而不是每个服务商都新建一个分类。
