# 开发者 API Key

开发者 API Key 可以让可信工具通过开放 API 操作你的 Subscription Manager 账号。

## 什么时候使用 API Key

当你希望脚本、助手或自动化工具执行以下任务时，可以使用 API Key：

- 查询订阅。
- 分析支出。
- 查找重复订阅。
- 在确认后创建或更新订阅。
- 读取审计历史。

不要把 API Key 放进公开前端代码或共享提示词里。

## 创建 Key

1. 登录账号。
2. 打开 **设置**。
3. 进入 **开发者 API**。
4. 输入 Key 名称。
5. 创建 Key。
6. 立即复制完整 Key。

完整 Key 只显示一次。之后应用只会显示 Key 前缀。

## 撤销 Key

打开 **设置** > **开发者 API**，撤销不再使用的 Key。撤销后，使用这个 Key 的后续 API 请求都会失败。

## 限额

普通账号可用 Key 数量和每小时请求数可能低于 Premium 账号。开发者 API 页面会显示当前套餐、可用 Key 数量和每小时请求限额。

## 安全建议

- 每个集成单独使用一个 Key。
- 不再使用的 Key 及时撤销。
- 如果 Key 可能出现在日志、截图或提示词中，立即轮换。
- 如果部署支持 scoped key，只做分析的 Agent 优先使用只读 Key。
