# 常见问题

## 我的数据存在哪里？

不登录时，数据保存在当前设备的浏览器里。如果你登录且云同步可用，数据可以同步到你的账号下。

## 必须注册账号吗？

不需要。不登录也可以本地记录订阅。需要云同步、账号设置、Premium 状态、依赖托管同步的提醒，或开发者 API Key 时再登录。

## AI 录入会保存我的截图或粘贴文本吗？

不会。你提交的文本或截图只会为了这次识别发送给配置的模型供应商一次。Subscription Manager 不保存、不记录这些内容。

## 为什么切换币种后总支出会变化？

概览会把订阅金额换算成当前基础币种。汇率会变化，所以换算后的总支出可能变化，但每个订阅保存的原始金额不会变。

## 离线汇率或汇率过期是什么意思？

这表示应用无法刷新实时汇率，正在使用缓存汇率或备用汇率。订阅原始金额不受影响。

## 为什么没有收到提醒？

检查 Bark 设置是否已保存、device key 是否正确、该订阅是否启用了通知、下一次付款日期是否有效。如果托管提醒依赖云同步，也需要确认你已经登录。

## 如何把数据移动到另一个浏览器？

在旧浏览器使用 **导出数据**，在新浏览器使用 **导入数据**。

## 可以撤销修改吗？

手动编辑可以再次打开订阅修改。AI 操作在确认后可能会显示即时撤销提示，请及时使用。
