# 币种和汇率

每个订阅都有自己的原始币种。概览区域可以按你选择的基础币种展示总支出。

## 支持的币种

Subscription Manager 支持 CNY、USD、EUR、JPY、GBP、AUD、CAD、CHF、HKD 和 SGD。

## 选择基础币种

在概览区域使用币种选择器。应用会把订阅金额换算成这个币种，用于展示月度和年度总支出。

切换基础币种只影响概览和图表，不会修改每个订阅保存的原始币种。

## 月度和年度视图

月度视图会估算每个订阅的月均成本。年度视图会估算年化成本。

| 订阅周期 | 月度视图 | 年度视图 |
| --- | --- | --- |
| 每月 | 月费 | 月费 x 12 |
| 每年 | 年费 / 12 | 年费 |
| 自定义 | 按间隔天数折算 | 按间隔天数折算 |

## 汇率状态

应用会尽量刷新汇率，并保留缓存。如果无法刷新实时汇率，你可能会看到离线或汇率过期提示。此时仍会展示总支出，但可能使用缓存汇率或备用汇率。

如果需要精确核算，请以每个订阅保存的原始币种和金额为准。
