# 订阅管理

订阅记录适合管理流媒体、软件、云服务、会员、域名，或者任何你想持续追踪的周期性付款。

## 添加订阅

1. 点击 **添加订阅**。
2. 输入订阅名称。
3. 选择分类，或添加新分类。
4. 输入金额和货币。
5. 选择付款周期：每月、每年或自定义。
6. 输入上次付款日期。
7. 选择是否为这个订阅启用通知。
8. 保存订阅。

应用会根据上次付款日期和付款周期计算下一次付款日期。

## 付款周期

| 周期 | 适合什么 |
| --- | --- |
| 每月 | 每个月续费一次的服务 |
| 每年 | 每年续费一次的服务 |
| 自定义 | 每隔固定天数续费的服务 |

自定义周期需要填写间隔天数。例如每 45 天扣费一次，就填写 `45`。

## 编辑订阅

从列表打开一个订阅，然后选择编辑。你可以修改名称、分类、金额、币种、付款周期、上次付款日期、自定义周期和通知设置。

修改上次付款日期或付款周期后，下一次付款日期会随之更新。

## 筛选和排序

在概览区域可以：

- 在月度和年度总支出之间切换。
- 切换基础币种。
- 按分类筛选。
- 按金额、名称、分类、续费日期或创建时间排序。
- 切换升序或降序。

## 删除订阅

打开订阅详情后，可以删除不再需要追踪的订阅。删除会把记录从你的列表中移除。

如果你通过开放 API 使用应用，也可以用取消或暂停状态保留历史；在应用界面中，删除是主要的移除操作。
