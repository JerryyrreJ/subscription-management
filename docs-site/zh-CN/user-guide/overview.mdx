# 使用指南

Subscription Manager 用来记录你的周期性付款：订阅了什么、多久续费一次、下一次什么时候付款，以及按你选择的币种统计大概要花多少钱。

## 你可以管理什么

- 订阅名称、金额、币种、分类、付款周期和上次付款日期。
- 按基础币种展示月度和年度总支出。
- 用分类整理不同服务。
- 用 AI 录入提议新增、修改、删除和批量更新。
- 通过 Bark 接收续费提醒。
- 通过 JSON 导入导出做本地备份。
- 登录后在多设备之间同步。
- 创建开发者 API Key 给可信自动化工具使用。

## 默认本地优先

不登录也可以使用应用。此时订阅、分类、设置和备份数据保存在当前浏览器里。

登录后，如果部署方启用了云同步，你的数据可以同步到账号下并在多设备间使用。

## 隐私基础

- 手动记录的数据默认留在本地，除非你登录并同步。
- AI 录入只会把你本次提交的文本或截图发送给配置的模型供应商。
- Subscription Manager 不保存 AI 录入的原始文本或截图。
- 开发者 API Key 应像密码一样保管。
