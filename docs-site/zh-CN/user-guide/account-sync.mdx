# 账号和同步

Subscription Manager 是本地优先应用。你可以不登录直接使用，也可以在云同步可用时登录，让数据在多个设备之间同步。

## 不登录使用

数据会留在当前浏览器。这种方式简单、私密，但如果清理浏览器数据或更换设备，数据可能不会自动带过去，建议先导出备份。

## 登录

使用 **登录后同步** 或账号菜单登录。登录后，应用可以把订阅、分类和通知设置同步到你的账号下。

## 同步状态

用户菜单会显示同步状态，例如同步中、上次同步时间、同步失败或从未同步。需要手动触发时，可以使用 **立即同步**。

## 账号设置

打开 **设置** > **账号** 可以管理：

- 昵称。
- 邮箱。
- 密码。
- 永久注销账号。

账号设置只会在登录后显示。

## 注销账号

打开 **设置** > **账号** > **危险操作** 可以永久注销账号。注销会删除登录信息、个人资料、云端订阅、分类、通知设置、API Key、Premium 状态，以及当前设备上属于该账号的本地数据，而且无法撤销。

注销不会自动产生退款。如果账号存在付款记录，应用仍会在必要期限内保留付款邮箱和必要的 Stripe 交易信息，用于财务核对、退款处理、支付争议及适用法律法规要求，不用于营销。

应用无法远程清除其他设备浏览器中已经保存的本地副本。如有需要，请同时在其他设备上清除本站数据。

## 本地数据和云端数据

登录后，应用会把本地数据按你的用户账号隔离，并在配置允许时同步到云端。如果数据很重要，在更换账号或设备前建议先导出备份。
