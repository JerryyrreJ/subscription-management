# Account and sync

Subscription Manager is local-first. You can use it without signing in, or sign in to sync data across devices when cloud sync is available.

## Without an account

Your data stays in the current browser. This is simple and private, but it means clearing browser storage or switching devices can leave data behind unless you export a backup.

## Sign in

Use **Login to sync** or the account menu to sign in. After signing in, the app can sync subscriptions, categories, and notification settings with your account.

## Sync status

The user menu shows sync status such as syncing, last synced time, sync failed, or never synced. Use **Sync now** if you want to trigger a sync manually.

## Account settings

Open **Settings** > **Account** to manage:

- Nickname.
- Email address.
- Password.

Account settings are only visible when signed in.

## Local data and cloud data

When you sign in, the app scopes local data to your user account and syncs with the cloud when configured. Export a backup before major account or device changes if the data is important.
