# Categories

Categories help you group subscriptions for filtering and spending analysis.

## Choose a category

When adding or editing a subscription, choose a category from the list. Category names appear in filters, charts, and subscription cards.

## Add a custom category

1. Open **Settings**.
2. Go to **Categories**.
3. Add a new category name.
4. Use it when adding or editing subscriptions.

You can also add a category from the subscription form when choosing a category.

## Reorder categories

Open **Settings** > **Categories** and drag categories into the order you prefer. This order affects how categories appear in category selectors.

## Delete or restore

You can remove custom categories or restore default categories from category settings. If a category is used by subscriptions, check the confirmation carefully so existing subscriptions do not become harder to organize.

## Tips

- Keep category names short and consistent.
- Use broad categories such as Streaming, Productivity, Cloud, Finance, or Shopping.
- Prefer one category per purpose instead of one category per vendor.
