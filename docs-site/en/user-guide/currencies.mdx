# Currencies

Each subscription has its own currency. The dashboard can show totals in a selected base currency.

## Supported currencies

Subscription Manager supports CNY, USD, EUR, JPY, GBP, AUD, CAD, CHF, HKD, and SGD.

## Choose the base currency

Use the currency selector in the dashboard overview. The app converts subscription amounts into that currency for monthly and yearly totals.

Changing the base currency affects dashboard totals and charts, but it does not change the original currency saved on each subscription.

## Monthly and yearly views

Monthly view estimates the monthly equivalent of each subscription. Yearly view estimates the annual equivalent.

| Subscription period | Monthly view | Yearly view |
| --- | --- | --- |
| Monthly | Monthly amount | Monthly amount x 12 |
| Yearly | Yearly amount / 12 | Yearly amount |
| Custom | Prorated by interval days | Prorated by interval days |

## Exchange rate status

The app refreshes exchange rates when possible and keeps cached rates. If live rates cannot refresh, you may see an offline or stale-rate message. In that case, totals are still shown, but they may use cached or fallback rates.

For exact accounting, check the original subscription currency and amount.
