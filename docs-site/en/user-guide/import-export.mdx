# Import and export

Use import and export to back up your data or move it between browsers and deployments.

## Export data

1. Open the user menu or **Settings** > **General**.
2. Choose **Export data**.
3. Save the downloaded JSON file somewhere private.

The export includes subscriptions and category data that the app can import later.

## Import data

1. Open the user menu or **Settings** > **General**.
2. Choose **Import data**.
3. Select a JSON export file.
4. Review the import preview.
5. Confirm the import.

The import flow validates the file before applying changes.

## Keep backups private

Export files can include names, payment amounts, renewal dates, categories, and notification preferences. Treat them like personal finance data.

## When to export

- Before switching browsers or devices.
- Before clearing browser data.
- Before testing a self-hosted deployment.
- Before making a large batch of edits.
