# Reminders

Reminders help you notice upcoming renewals before a subscription charges again.

## What you need

Renewal reminders use Bark. You need:

- A Bark server URL.
- A Bark device key.
- A reminder timing, such as how many days before renewal to notify you.

If you use the hosted app, reminders must also be enabled by the deployment. If reminders are unavailable, you can still track renewal dates manually in the app.

## Configure reminders

1. Open **Settings**.
2. Go to **Notifications**.
3. Paste a Bark URL or enter your Bark server and device key.
4. Choose the reminder timing.
5. Save the settings.

The app can extract the device key from a Bark example URL.

## Per-subscription toggles

Each subscription has its own notification setting. Turn it off for subscriptions that you do not want reminders for.

## Test and troubleshoot

If reminders do not arrive:

- Check that the Bark server URL is reachable.
- Check that the device key is correct.
- Check that the subscription has notifications enabled.
- Check that the next payment date is valid and in the future.
- Check that you are signed in if the hosted reminder flow requires cloud sync.
