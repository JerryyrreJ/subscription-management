# User guide

Subscription Manager helps you keep a personal list of recurring payments: what you pay for, how often it renews, when the next payment is due, and how much it costs in your chosen currency.

## What you can manage

- Subscriptions with amount, currency, category, billing period, and last payment date.
- Monthly and yearly totals in a base currency.
- Categories for organizing services.
- AI-assisted creates, updates, deletes, and batch updates.
- Renewal reminders through Bark.
- Local backups through JSON import and export.
- Optional account sync across devices.
- Developer API keys for trusted automations.

## Local-first by default

You can use the app without an account. In that mode, your subscriptions, categories, settings, and backups stay in the browser on the current device.

Signing in enables cloud sync if the hosted service is configured. Your local data can then sync across devices under your account.

## Privacy basics

- Manual data stays local unless you sign in and sync.
- AI capture sends only the text or screenshot you submit for that request to the configured model provider.
- AI capture content is not stored by Subscription Manager.
- Developer API keys should be treated like passwords.
